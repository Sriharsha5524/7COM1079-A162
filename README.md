# 7COM1079 Team Research and Development Project  
## Analysis of Olympic Medal Performance: Host vs Non-Host Countries

This repository contains all the materials for our group project investigating whether hosting the Summer Olympics is associated with higher average medal totals. The project explores historical Olympic performance data and applies statistical testing to determine if host nations show a measurable advantage.

## Project Overview  
The aim of this project is to compare total Summer Olympic medals for countries that have hosted the Games at least once with those that have never hosted. This comparison is carried out using descriptive statistics, data visualisation, and a two-sample t-test.

---

## Research Question  
**The Relationship between Hosting the Summer Olympic Games and Total Medal Success**
